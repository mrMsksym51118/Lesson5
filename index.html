Виды CSS селекторов:

  • по тегам:

    • h1 - заголовки на странице приобретали стиль данного класса

  • по классу:

    • сначала ставим точку, затем сразу, без пробела,пишем имя класса, ну а затем в фигурных скобках стиль

  • по id:

    • данный селектор применяется если необходимо выделить один единственный элемент

  • по нескольким классам:

    • через точку без пробелов пишутся два разных класса. Стили будут применяться ко всем элементам, которые одновременно имеют эти два класса

  • контекстные:

    • разделяются пробелом. Такие селекторы используют для того, чтобы применить стили к элементу, только если он вложен в нужный элемент

  • соседние:

    • выделяет последующий элемент. Записываеться спомощью знака +

  • родственные:

    • выделяет последующие элементы. Записываються с помощью знака ~

  • дочерние:

    • записываются с помощью знака > и распространяются только на элементы первого уровня вложенности. На элементы вложенные в другие элементы не распространяются

  • атрибутов:

    • чаще всего используются при работе с формами. Записываются с использованием []

  • глобальные:

    • селектор * выберет все элементы. Его также можно использовать для выделения дочерних элементов

  • псевдоклассы:

    • псевдокласс отрицания :not бывает очень полезным. Например, когда необходимо выбрать все div, за исключением того, который имеет id container
